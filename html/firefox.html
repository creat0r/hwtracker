How to set up Firefox

  1. Open Mac Mail. (If this is the first time you need to set it up by using pop.gmail.com and smtp.gmail.com). Go to preferences > general > default email reader to Firefox
  2. On the menu bar, click on the Firefox menu and select Preferences.
  3. Select the Applications panel.
  4. Search for the Content-type: mailto and select it.
  5. Click on the Actions column in the mailto row, select gmail.
